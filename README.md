# Тренировки — Telegram Mini App

Приложение для учёта тренировок: создание планов, запись подходов, весов и повторений по дням.

## Возможности

- **Профиль** — у каждого пользователя свой (через Telegram)
- **Дни тренировок** — выбор дней недели (Пн, Вт, Ср...)
- **План на неделю** — составление тренировок для каждого тренировочного дня
- **Журнал** — запись подходов, весов и повторений по дням
- Всё редактируется и сохраняется локально

## Запуск локально

Открой `index.html` в браузере или используй локальный сервер:

```bash
npx serve .
```

Или с Python:
```bash
python -m http.server 8000
```

## Подключение к Telegram

1. Создай бота через [@BotFather](https://t.me/BotFather)
2. Задеплой приложение на HTTPS (Vercel, Netlify, GitHub Pages и т.п.)
3. В BotFather выбери бота → Bot Settings → Menu Button
4. Укажи URL своего приложения
5. Либо через команду `/setmenubutton` — передай URL
